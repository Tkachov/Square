# Класс Timer

## Общая информация

Класс Timer может использоваться для вызова функции с некоторым интервалом. Он наследует класс [Entity](04_Entity.md), но ничего не отрисовывает на экране.

Timer остановится после того, как вызовет функцию **N** раз, если **N** положительно (ни отрицательно, ни равно 0).

## Functions

##### `Timer(Controller*, int interval, int N, functor* f)`
Конструктор, создаёт Timer, который вызывает **f** каждые **interval** кадров **N** раз (или не перестаёт вызывать её, если **N** равно нулю или отрицательно).  

----
##### `void redraw(int x, int y)` [наследуется из [Entity](04_Entity.md#void-redrawint-x-int-y)]
Ничего не делает.  

----
##### `void update(int x, int y)` [наследуется из [Entity](04_Entity.md#void-updateint-x-int-y)]
Обновляет состояние объекта и вызывает функцию, если требуется.
